○サンプルプログラムについて

	本フォルダは、GR-FILEを使用するために参考となるサンプルプ
	ログラムとなります。
	本サンプルプログラムの動作には、弊社製USBマスストレージ統
	合キットが必要です。

	本サンプルプログラムは理解しやすくする為に全角文字を使用し
	ています。
	ツールによっては全角文字が使用できない場合がありますのでご
	注意ください。

	本サンプルプログラムは、初期処理手順やデバイス接続時の処理
	について参考としていただくことを目的としていますので、コン
	パイル等については保障いたしかねますことをご了承ください。
	また本サンプルプログラムでは、ポーティング対象となっている
	関数やパラメータに関して、テスト用として実装しております。
	ドキュメントを参照し、お使いのシステムに合わせた処理を実装
	ください。

	※その他、詳細につきましては各モジュールのドキュメントを参
	  照ください。

○サンプルプログラム
	vos
		GR-VOS Version1.xxを使用したサンプルディレクトリです。
		GR-USBにはHOSTIIを使用します。
	vos2.xx
		GR-VOS Version2.xxを使用したサンプルディレクトリです。
		GR-USBにはHOST#を使用します。
	
	上記ディレクトリには以下の様なサンプルがあります。

	gr_file_sample
		弊社製USBマスストレージキットを使用した場合のサンプル
		プログラムです。
		また、コンソール入出力にシリアルを使用しますので使用す
		る環境に合わせたシリアルドライバを用意してください。

	gr_file_sample_no_console
		弊社製USBマスストレージキットを使用した場合のサンプル
		プログラムです。
		文字列出力にシリアルを使用しますので使用する環境に合わ
		せたシリアルドライバを用意してください。
		（文字列出力を行わなくすることも可能です）
		コンソール出力を使用しない場合はサンプルプログラム
		の動作をデバッガで参照し、GR-FILE APIの戻り値など
		を確認していただく必要があります。
